## Product Overview

PanoVR Demo is a web-based panoramic VR viewer that displays 360-degree street view imagery with interactive navigation. The application combines 2D map visualization with immersive VR panorama viewing, supporting both standard browser viewing and WebXR for VR headsets.

### Key Features

- Interactive 2D map with panorama location markers
- 360-degree panorama viewer with hotspot navigation
- WebXR support for VR headsets (Quest, etc.)
- Graph-based navigation between connected panorama nodes
- Gamepad/controller support for VR navigation
- GPS-aware next/prev navigation based on spatial relationships
